Themes

Customize appearence of elements

Install SASS loader

Following Nuxt's documentation at nuxtjs.org

# yarn
yarn add --dev sass sass-loader@10

# npm
npm i -D sass sass-loader@10

Create app.scss

// ./assets/scss/app.scss

// Custom variables, refer _settings.scss
$color-primary: #00d4fa;
$font-family: 'Inter', sans-serif;
$html-font-size: 10px;
$body-font-size: 14px;

// Required
$assets-path: "@bigin/bigin-ui-ds/src/assets";
@import "@bigin/bigin-ui-ds/src/scss/bigin-ui";

Update configuration

Adding app.scss to to config file

// nuxt.config.js

export default {
  ...,
  css: ['@/assets/scss/app.scss'],
  ...
}
